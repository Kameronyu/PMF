Create `tools/asset/probe.py`. Two modes via argv:

1) **Single-image probe** (`probe.py <raw_path> <work_path>`): open with `ImageOps.exif_transpose(Image.open(raw_path))`, read w/h, compute `imagehash.phash`, thumbnail to (1400,1400), save `work_path` as `JPEG quality=88` (convert RGB first). Print JSON to stdout. Seed (paste, then extend):

```python
import sys, os, json
from PIL import Image, ImageOps
import imagehash

def probe(raw_path, work_path, max_edge=1400):
    im = ImageOps.exif_transpose(Image.open(raw_path))
    w, h = im.size
    fmt = (im.format or os.path.splitext(raw_path)[1].lstrip('.')).lower()
    long_edge = max(w, h)
    min_safe_use = "hero" if long_edge >= 3000 else "section" if long_edge >= 1500 else "thumb"
    phash = str(imagehash.phash(im))
    rgb = im.convert("RGB")
    rgb.thumbnail((max_edge, max_edge))
    os.makedirs(os.path.dirname(work_path), exist_ok=True)
    rgb.save(work_path, "JPEG", quality=88)
    return {"w": w, "h": h, "aspect": f"{w}:{h}", "format": fmt,
            "bytes": os.path.getsize(raw_path), "phash": phash,
            "dup_of": None, "min_safe_use": min_safe_use}

if __name__ == "__main__":
    if len(sys.argv) >= 4 and sys.argv[1] == "--dedupe":
        # mode 2 below
        ...
    else:
        raw_path, work_path = sys.argv[1], sys.argv[2]
        print(json.dumps(probe(raw_path, work_path)))
```

2) **Dedupe pass** (`probe.py --dedupe <records.json>`): given a JSON array of `{id, phash, technical:{w,h}}` (or `{id, phash, w, h}`), compute pairwise Hamming distance between phashes (`imagehash.hex_to_hash(a) - imagehash.hex_to_hash(b)`); for any pair with distance **< 5** (threshold ~5/64), mark the **lower-resolution OR softer** one `dup_of: <winner_id>` (winner = larger w*h; tie -> keep first). Print the updated array as JSON to stdout. This is the across-the-set comparison the single-image mode can't do.

Grounding/never-fabricate: on any PIL error, print the error to stderr and exit non-zero (the Node caller refuses to fabricate technical{}). Do NOT swallow exceptions.

NOTE for the executor: do not commit `assets/` (runtime output) — add `assets/` to `.gitignore` if not present. probe.py itself is committed.
