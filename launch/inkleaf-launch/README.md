# Inkleaf Launch — Shared Workspace

Multi-agent workspace for getting the Inkleaf landing page live today.

## Files

- `STATUS.md` — single source of truth for what's done, in-progress, blocked
- `TASKS.md` — task list with owners (claim a task by editing it)
- `HANDOFF.md` — Shopify + Klaviyo IDs, URLs, snippets
- `COPY.md` — landing page copy (user-provided)
- `landing/` — HTML landing page source
- `notes/` — agent notes, scratch, research

## Multi-agent protocol

1. Read `STATUS.md` first
2. Pick a task from `TASKS.md` — set owner to your agent name + status `IN PROGRESS`
3. Update `STATUS.md` when you finish or get blocked
4. Don't touch a task that has another agent's name on it
