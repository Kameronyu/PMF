**Color palette:** Descriptive only — copy references "White" and "Crystal" device colorways; the LAMY Special Set emphasizes translucency and "skeleton" visibility. Brand tone is monochrome-minimalist. No hex values available.

**Typography:** Not directly named in copy. Brand uses clean, modern lowercase/mixed-case section headers ("Built to Laaaast," "Organize Magically") suggesting playful sans-serif at headings, functional body text. The deliberately misspelled "Laaaast" indicates intentional tonal casualness.

- Hand-with-pen / stylus: multiple copy references to "natural writing feel," "writing experience," and pen close-ups
- Product-on-desk: Manta positioned as "desktop use" device with "Stable Foot Pads — Write more comfortably on desktop"

- Community / UGC: "#SupernoteManta in real life" gallery section; "#NomadLife" hashtag section on Nomad PDP
- Product isolation: LAMY Special Set copy describes translucent body visibility, "skeleton body" aesthetic — implies product-isolation photography

**Density:** Copy architecture is information-dense on PDPs (multiple section headers per page, feature-grid format), but individual section copy is concise. Use-case LPs (Students, Healthcare, etc.) follow a simpler feature-block + testimonial structure.

**Motion:** Multiple video ads (11 active ads, majority video: 0:15, 0:28, 0:29, 0:36, 0:44 durations). PDPs include "WATCH VIDEO" CTA. Static images in Ads 2, 4, 5, 6.
