## Delay Timeline Reconstructed (chronological)

| Date | Status |
|---|---|
| Oct 2 2018 | Campaign launches. Goal: $50k. Funded in 28 minutes. |
| ~Oct–Nov 2018 | Campaign runs; $300k+ raised in first week; ~$530k+ by day 30. |
| ~Nov 2018 | Campaign closes (approx 60-day IGG campaign window implied). |
| Dec 2018–Jan 2019 | Liliputing reported "Shipments were expected to begin in January" — this was the original projected date. Wikipedia also cites "first units shipped January 2019" — likely refers to a small advance/developer batch only, not mass fulfillment. |
| ~Spring 2019 | Delivery slips from Jan 2019 to June 2019. |
| Summer 2019 | June date missed. New date: August 2019. |
| ~Aug–Sep 2019 | August missed. New date: Fall 2019. Company "expecting to ship the first batch of units around November 2019." |
| ~Nov 2019 | November date missed. New date: Spring 2020. |
| ~Mar–Apr 2020 | Spring/March 2020 date missed (compounded by COVID-19 supply chain disruption). |
| ~Jun 2020 | Official update: "We are still on track for a Fall 2020 ship date." Factory producing 50 pre-production samples. |
| Oct–Nov 2020 | First production run ships. Hands-on reviews published (GeekDad Oct 28, TechCrunch Oct 27, Engadget Nov 2, Tarina Deaton Oct 28). |

Sources:
- liliputing.com (Oct 2 2018): "Shipments were expected to begin in January."

- getfreewrite.com/pages/freewrite-traveler-updates (June 2020 update verbatim): "The factory is going to be creating 50 pre-production samples of Traveler over the next month!" / "We are still on track for a Fall 2020 ship date."
