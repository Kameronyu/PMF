# Finder Brief — Faith / Devotional Market · Competitive Set

You build the competitive-set roster for ONE market so analyzer agents can study each
competitor. **One job: find competitors, stratify, rank.** You do NOT analyze in depth.

## Market

- **Product (fixed):** a foldable, programmable e-ink tablet. $900 target retail.
  Pipeline B — product is held constant; this market is one candidate niche.
- **Niche:** faith / devotional people — Bible readers, prayer practitioners, sermon
  note-takers, the devotional / journaling-Bible community (Christian).
- **Transformation:** a focused, present devotional life — undistracted time in
  Scripture and prayer.

## What counts as a competitor

The competitive set is **everything the buyer could use to get focused devotional
time — not just e-ink tablets.** Substitutes count. Scoping to one product category
undercounts the real competition. Four strata:

1. **Physical / analog** — journaling Bibles, wide-margin Bibles, prayer journals,
   faith / devotional planners, AND premium / heirloom leather Bibles (Schuyler,
   Cambridge, Allan's etc. — these set the price ceiling; capture them deliberately).
2. **Dedicated apps** — Bible apps, prayer apps, Christian wellness / meditation apps.
   Free and paid. Include the dominant free one and the heavily-marketed paid ones.
3. **General tablets repurposed** — iPad / Android tablets + Bible-study software
   ecosystems (Logos, Olive Tree, Accordance).
4. **E-ink dedicated** — e-ink / paper tablets. **Check whether ANY targets faith.**
   If none do, that is itself the finding. Include the major incumbents (reMarkable,
   Daylight, Boox, Kindle Scribe, Supernote) as repurposable substitutes.

## Output

Write `runs/eink-tablets/markets/faith/competitive-set.md`:

- Per stratum: 5–8 entries, **ranked** by market relevance + size / scale.
- Each entry: name · URL · one-line what-it-is · one-line why-it-matters (size,
  price signal, or relevance) · stratum.
- Mark the **2–3 must-analyze entries per stratum**.
- For physical: include at least one cheap anchor ($25–60 journaling Bible) AND one
  premium anchor ($100–400+ heirloom Bible) — the price spread is a key finding.
- Note any entry already covered by the existing eink run (`runs/eink-tablets/brands/`).
- Answer explicitly: **does any e-ink player target faith?** (yes / no + evidence).

Rank, don't analyze. Keep it factual and dense.
