# PMF

Ecom research / testing framework. DR marketing system that maps markets, identifies opportunity, designs tests.

**Entry point:** [`handoff.md`](handoff.md) — read first. It points to the rest in order.

## Layout

- `definitions.md` — locked vocabulary
- `workflow.md` — phase structure + per-phase research questions (Phase −1 → 8); the run spine
- `capability_inventory.md` — ~20 atomic capabilities, tagged
- `handoff.md` — session entry point, what's locked, what's open, what's next
- `map/` — persistence layer design (foundational Under #1)
- `runs/` — manual research runs (e.g. `runs/eink-tablets/` — Phase 0 competitor map)
- `agents/` — agent specs (not yet written; gated on map layer)
