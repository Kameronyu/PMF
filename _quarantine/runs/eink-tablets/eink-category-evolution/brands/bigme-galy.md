## Mechanism / UM
- mechanism — how/why it works: Gallery 3 uses E Ink's Advanced Color ePaper (ACeP) — four pigment particles (cyan, magenta, yellow, white) at each pixel, producing the full color gamut at every pixel instead of relying on a color filter array over a B/W panel.
- problem-mechanism (causal story): Previous color e-ink (Kaleido / Kaleido Plus / Kaleido 3) uses a CFA over B/W E Ink, which dims the panel and limits color saturation. Gallery 3 removes the filter and puts color in the ink itself, recovering brightness and saturation — but at the cost of slower refresh.
- UM: **Feature-UM — "Gallery 3 (ACeP four-particle ink)"**, positioned as the next-generation upgrade over Kaleido within the recognized color-e-ink-tablet category. Buyer still recognizes the product as a color e-ink tablet; Gallery 3 is the upgrade lever. NOT a Product-UM (no recategorization) and NOT a Problem-UM (Bigme doesn't blame a unique cause for color e-ink limitations beyond the implicit "Kaleido is dim").

## Evolution Anchor

- **Launch year:** 2022 (KS live Nov 16 – Dec 31, 2022; first ships January 2023).

- **Main differentiation claim AT LAUNCH (verbatim):** "World's First Color E Ink Gallery™ 3 Tablet."

- **What pre-existing claims competed with:**
  - Boox Nova Air C / Tab Ultra C / Note Air2 Plus C (Kaleido / Kaleido Plus, 2021–2022) — "color e-ink productivity tablet."
  - PocketBook Color / Viva (Kaleido, 2020–2022) — "color e-reader."
  - Bigme's own inkNote Color (Kaleido Plus, Jul 2022, four months prior) — "World's First Color E-Ink Tablet w/ Cameras."
  - Reinkstone R1 (DES color, 2021) — "Ultimate True Color DES E-Paper Tablet."
  - The pre-existing pattern was uniformly: "color e-ink + Android + stylus, Kaleido panel." Galy's claim attacks the panel.

- **New claim / mechanism / feature introduced — Gallery 3 as Feature-UM (yes):**
  The differentiator is unambiguously a **Feature-UM**. Gallery 3 / ACeP is a panel-tech upgrade *within the existing color-e-ink-tablet category*. Buyers do not have to recategorize the product (it's still "a color e-ink tablet"); they are being upgraded on the *how* (four-particle ink instead of CFA). The mechanism (ACeP four-pigment electrophoretic ink) and the outcome it produces (richer color, no dimming filter, full color gamut at every pixel) are both new. This is the textbook Stage-4 differentiation move: alternative Feature-UM inside an established Product-UM. Bigme did NOT introduce a new transformation, a new niche, a new buyer, or a new problem-cause. The entire wedge is "better panel tech."

- **Why did Bigme launch TWO color tablets in 2022 at different tech stacks?**
  Bigme launched inkNote Color (Kaleido Plus, 10.3", Jul 2022) and Galy (Gallery 3, 8", Nov 2022) within four months. Speculation, but the copy supports it:
  1. **Hedge on E Ink's tech rollout.** Gallery 3 was brand new and unproven (slow refresh, expensive panel, supply constrained). Kaleido Plus was mature, cheap, and proven. Bigme covered both bets so they could capture buyers regardless of which color tech the market preferred.
  2. **Form-factor split, not direct overlap.** inkNote Color is a 10.3" full-size productivity slate (productivity / note-taking / PDF), positioned against Boox Note Air2 Plus C and reMarkable. Galy is an 8" reading/sketching tablet (one-handed reading, comic / magazine consumption, color-fidelity-first use cases) — positioned against the Boox Nova Air C / PocketBook Color cell.
  3. **Two "world's firsts" in one year is a PR / KS playbook move.** Bigme is a small OEM that uses Kickstarter as a launch + pre-order channel, not as a demand-validation channel. Each "World's First X panel tech" headline buys a fresh wave of tech-press coverage (Good e-Reader, Liliputing, Notebookcheck, CNX) and a fresh KS funnel. The cost of running a second KS within the same year is low; the marginal PR value is high.
  4. **Different panels = different supply chains.** E Ink Corp. allocates Gallery 3 separately from Kaleido. Running two products on two panels means Bigme isn't bottlenecked on either panel's yield.

- **Current evolution status (as of May 2026):**
  - Gallery 3 has since proliferated: **reMarkable Paper Pro (2024)** put Gallery 3 in front of a much larger audience; Readmoo **mooInk V (2025)** brought Gallery 3 to a foldable form factor; Boox Tab Mini C2 / Note Air3 C use Kaleido 3 still; Bigme followed up with inkNote Color+ on Kaleido 3.
  - Bigme Galy itself: still listed on Bigme store at $699; superseded as Bigme's flagship by the dual-screen e-ink+LCD smartphone announcements (2024 IFA, 2026 press releases) and by inkNote Color+ on Kaleido 3.
  - Galy's "World's First Gallery 3 tablet" claim has fully decayed — Gallery 3 is now a recognized panel option, not a wedge. reMarkable's Paper Pro made Gallery 3 a mass-market reference point, which collapsed Galy's Feature-UM into a commodity spec.
  - Category-level outcome: the Gallery 3 vs Kaleido 3 split is now stable. Kaleido 3 won the "reading / e-note / text-first" cell (because of cheaper panel, faster refresh, near-mono text). Gallery 3 won the "color-fidelity / drawing / image-first" cell (because of full gamut at every pixel). Bigme Galy is now an early-but-niche entrant in the Gallery 3 cell that lost mass-market ownership to reMarkable Paper Pro.
