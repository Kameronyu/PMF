## Key Specs as Stated in Campaign-Era Sources

- 6-inch E Ink screen (ambient light, no backlight)
- Full-size scissor switch keyboard, over 2mm key travel
- Dimensions: 12 x 5 x 1 inches / 302 x 127 x 25 mm
- Weight: ~1.8 lbs (0.8 kg) — later revised to 1.6 lbs in retail spec
- Battery: up to 30 hours continuous; 4 weeks at ~30 min/day
- Storage: capacity "exceeds one million pages"
- USB-C charging
- 2.4 GHz Wi-Fi
- Cloud sync: Postbox, Dropbox, Evernote, Google Drive
- Supports 30+ keyboard layouts and languages (campaign era); later spec says 60+ languages, 80+ layouts
- No copy/paste, no spellcheck, no third-party apps
- Secondary status screen for word count and work-duration milestones
- Markdown syntax support for basic formatting (headings, bullets, italics)
- No trackpad or mouse
- Document launcher / folder navigation feature (new vs original Freewrite)
- WASD arrow key functionality for cursor movement (improvement over original)

Source: newatlas.com (Oct 16 2018), liliputing.com (Oct 2 2018), thenextweb.com (Oct 2 2018),
        digitaltrends.com (Oct 3 2018), ubergizmo.com (Oct 3 2018)
