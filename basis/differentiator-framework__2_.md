# Differentiator Framework — Case Study Reference

## Definitions

**Market** = person + transformation. If the core human driver changes, it's a new market.
**Mech (UM)** = unique way of delivering the transformation. Problem UM = why they have the problem. Solution UM = how your product fixes it.
**Angle** = emotional frame / doorway into the SAME transformation. If the transformation changes, that's a new market, not a new angle.
**Offer** = commercial wrapper (bundle, price, guarantee, subscription, loss leader).
**Copy/VOC** = constant. Not a differentiator. Always write your best copy regardless.

## Levers Within Each Differentiator

**Market levers:** Avatar flip, transformation change, buyer ≠ user, new sub-avatar, avatar specificity
**Mech (UM) levers:** New product, proprietary naming, new feature, format change, problem UM, solution UM
**Angle levers:** Desire, pain, external blame, care signaling, belonging/identity, curiosity/secret
**Offer levers:** Bundle, price, guarantee, subscription, loss leader

## Key Patterns

1. Market changes are the biggest swings.
2. Buyer ≠ user is a repeating pattern (Nusk, Maya, Grüns) — opens uncontested markets, often allows higher pricing.
3. Problem UM + Solution UM compound (Shower Head, Hexclad).
4. Angle is the emotional frame within the SAME transformation. If the core human driver changes, you've changed markets.
5. No differentiator = failure (Hair Dryer).
6. Changing markets decreases sophistication (Nusk, Maya).
